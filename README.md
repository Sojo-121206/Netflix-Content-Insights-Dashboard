# Netflix Content Insights Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard built using a Netflix dataset. The dashboard focuses on analyzing content distribution, identifying growth trends, and understanding audience segmentation through effective data visualization.

## Key Features
- KPI indicators for total titles, movies, TV shows, and countries
- Comparative analysis of content types (Movies vs TV Shows)
- Country-wise distribution highlighting top contributing regions
- Year-wise trend analysis of content additions
- Rating-based audience classification
- Genre-based insights into content categories

## Tools and Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning and Transformation

## Dataset
- Netflix dataset (cleaned version)
- File used: `cleaned_netflix.csv`

## Objective
The objective of this project is to generate meaningful insights into:
- Content distribution across different categories
- Growth patterns over time
- Audience targeting based on ratings and genres
- Regional contribution to content production

## Future Enhancements
- Integration of user engagement data
- Predictive analytics for trend forecasting
- Advanced filtering and drill-through capabilities
